# 💎 Oyin & Co. Jewelries — E-Commerce Frontend

A modern and responsive jewelry e-commerce frontend built with HTML, CSS, and Vanilla JavaScript.

This project showcases a clean shopping experience for a jewelry brand, featuring categorized products, cart interactions, responsive layouts, and a modern UI structure focused on user experience and simplicity.
